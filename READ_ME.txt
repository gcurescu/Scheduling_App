•  C195 Scheduling Application

The purpose of this application was to give users/businesses the ability to add/remove/update
customer and appointment information from an established MySQL database.

•  Author: Gentillo Curescu
   Contact: gcuresc@wgu.edu
   Version: 1.0.0
   Date: 10/06/2021
   IDE: IntelliJ IDEA Community Edition 2021.1.1 x64
   JDK: Java SE 11.0.11
   JavaFX: JavaFX-SDK-11.0.2
   JDBC: mysql-connector-java-8.0.25

•  How to run the program

Open the project in Intellij and configure the paths for JavaFX and the MySQL
connector. Upon completion of the previous step run the project with
"Controller.mainLauncher.java" as the main class.
